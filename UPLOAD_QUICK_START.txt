🎯 QUICK REFERENCE

STEP 1: CREATE NEW CONTENT
   → LMS Admin > Programs > Select Program > Chapters > Select Chapter
   → Topics > Select Topic > Contents > [New Content Button]

STEP 2: FILL FORM
   Title ........................... Required (e.g., "Intro to Python")
   Content Type .................... Select type from dropdown
   
   📹 VIDEO FILE            → Upload: .mp4 .webm .ogv .avi .mov .mkv
   📄 PDF DOCUMENT          → Upload: .pdf
   🖼️ IMAGE                 → Upload: .jpg .png .gif .webp .bmp
   ⬇️ DOWNLOADABLE           → Upload: .zip .xls .doc .ppt .txt .csv
   
   🌐 YOUTUBE VIDEO         → Enter: YouTube URL (e.g., youtube.com/watch?v=...)
   📄 TEXT CONTENT          → Enter: Plain text
   🌐 HTML CONTENT          → Enter: HTML markup

STEP 3: UPLOAD FILE (for video/pdf/image/download types)
   → Click file input box
   → Select file from computer
   → File appears selected
   → Click [Add Content]

STEP 4: CONFIRM
   ✅ Success message appears
   → Redirects to content list
   → New content visible in table

📋 SUPPORTED FORMATS & LIMITS

VIDEOS
  Formats: MP4, WebM, OGV, AVI, MOV, MKV
  Max Size: 50 MB
  Use for: Lectures, demonstrations, walkthroughs

PDFs
  Format: PDF only
  Max Size: 20 MB
  Use for: Textbooks, guides, research papers

IMAGES
  Formats: JPG, PNG, GIF, WebP, BMP
  Max Size: 10 MB
  Use for: Diagrams, charts, infographics, screenshots

DOWNLOADS
  Formats: ZIP RAR 7Z XLS XLSX DOC DOCX PPT PPTX TXT CSV
  Max Size: 30 MB
  Use for: Datasets, templates, spreadsheets, presentations

❓ TROUBLESHOOTING

Q: File upload shows error "File type not allowed"
A: Check file extension. For videos, use .mp4 or .webm (not .avi with certain codecs)

Q: File is rejected as "too large"
A: Split into smaller files or compress:
   • Videos: Use HandBrake or FFmpeg for compression
   • PDFs: Use online PDF compressor tools
   • Images: Use TinyPNG or ImageOptim

Q: Uploaded file won't play/open
A: Verify file format is correct:
   • Videos: Test in VLC Player first
   • PDFs: Open in Adobe Reader
   • Use correct file for content type

Q: File disappears after saving
A: Check browser console for errors. File path might not be stored.
   Contact system admin for database verification.

💡 TIPS & BEST PRACTICES

✓ Use descriptive titles for content
  BAD: "Lecture 5"
  GOOD: "Introduction to Python Variables and Data Types"

✓ Organize content in logical order
  Use the sequence number to control student view order

✓ Keep videos to reasonable length
  Recommendation: 5-15 minutes per segment (easier for mobile users)

✓ Compress videos before uploading
  Reduces storage, faster streaming

✓ Test all uploaded files
  Click on content preview to verify display/playback

✓ Provide descriptions
  Help students understand content before opening

📁 FILE STORAGE

Files are automatically stored in:
  instance/uploads/content/
    ├── videos/        (all video files)
    ├── pdfs/          (all PDF files)
    ├── images/        (all image files)
    └── downloads/     (all downloadable resources)

Filenames are automatically timestamped for uniqueness:
  Example: 20260329_143025_lecture.mp4
