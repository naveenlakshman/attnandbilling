Import Invoices from CSV

Instructions
Step 1: Download Template

Click the button below to download the CSV template with the correct column headers and example data.

Download CSV Template
Step 2: Fill in Your Data

Open the CSV file in Excel or a text editor and add your invoice data. Keep all columns as provided in the template.

Invoice Number Format: Use GIT/B/1, GIT/B/2, GIT/B/3, etc. for invoice numbers (not INV001, etc)
Step 3: Upload and Import

Upload the completed CSV file using the form below. The system will validate all data before importing.

Important:
  • All columns must be present (even if some are empty)
  • invoice_date must be in YYYY-MM-DD format
  • student_id must exist in the database
  • created_by (user ID) must be a valid user ID
  • branch_id must be a valid branch (if provided)
  • discount_type must be: none, fixed, or percentage
  • installment_type must be: full or custom
  • status must be: unpaid, partially_paid, paid, or cancelled
CSV Column Reference
Column Name Type Required Description Example

status Text Yes Status: unpaid, partially_paid, paid, or cancelled pending
created_by Number Yes User ID who created the invoice 1
branch_id Number Yes Branch ID where invoice was created 1
