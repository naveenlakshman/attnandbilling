Import Invoices from CSV

Import invoices from a CSV file with all required details including student info, dates, discounts, and amounts.

Required columns:
  • invoice_number - Unique invoice identifier (format: GIT/B/1, GIT/B/2, etc.)
  • student_id - Student ID (must exist in database)
  • invoice_date - Invoice date (YYYY-MM-DD format)
  • subtotal - Amount before discounts
  • discount_type - Type: 'none', 'fixed', or 'percentage'
  • discount_value - Discount amount or percentage
  • discount_amount - Calculated discount amount
  • total_amount - Total after discounts
  • installment_type - Type: 'full' or 'custom'
  • notes - Additional notes (optional)
  • status - Status: 'unpaid', 'partially_paid', 'paid', or 'cancelled'
  • created_by - User ID who created the invoice
  • branch_id - Branch ID
