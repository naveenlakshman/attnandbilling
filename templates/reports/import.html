How to Import
  1. Download Sample: Click the sample link for the table you want to import
  2. Prepare Data: Update the sample file with your own data (keep column names)
  3. Select Table: Choose which table from the dropdown above
  4. Upload: Select your CSV file and click Import
  5. Check Results: View success/error count in the notification message
⚠️ Important:
  • Column names must exactly match the samples
  • Dates should be in YYYY-MM-DD format
  • Numbers without currency symbols (e.g., 5000, not $5,000)
  • Empty cells are allowed and become NULL
  • Failed rows are reported with error messages
  • Data is inserted in order, so relationships may fail if referenced data doesn't exist
