🔗 The Bridge

This assignment links your ERP system (which manages student batches) with your LMS (which manages learning programs).

It's the critical connection that determines student access.

📚 How It Works
  1. Select a batch (e.g., "CCOM Morning Batch")
  2. Select a program (e.g., "Office Automation")
  3. Set access dates
  4. Activate the assignment
  5. All students in that batch can now access that program
🎯 Access Control

Active:
Students can access the program

Inactive:
Access is blocked temporarily

Expired:
Access ends on specified date

💡 Tips
  • ✓ One batch can access multiple programs
  • ✓ One program can be assigned to many batches
  • ✓ Use dates to control access windows
  • ✓ Can be changed/deactivated anytime
  • ✓ Check student enrollment before assigning
